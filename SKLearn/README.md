  *  numpy (http://www.numpy.org/)
  *  matplotlib(https://matplotlib.org/)
  *  pandas (https://pandas.pydata.org/)
  *  scikit-learn (http://scikit-learn.org)

The documentation for these libraries are great, so keep that in mind!
